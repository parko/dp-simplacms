# dp-simplacms
Payment module for SimplaCMS

# Инструкция
Для установки модуля в CMS Siimpla:
1. Скопировать папку DeltaProsessing из внутрь папки payments.
2. В админке открыть Настройки - Оплата и создать новый вид оплаты. При этом
2.1 Выбрать DeltaProsessing в качестве платежного модуля
2.2 Указать параметры Идентификатор терминала и Секретный ключ (значения их получить из своего аккаунта в DeltaProsessing). 
2.3 Установить флажок Активен.
2.4 Остальные параметры в соответствии с конкретными предпочтениями.
После этого доступен новый модуль оплаты.
